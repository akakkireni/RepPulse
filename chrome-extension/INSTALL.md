# Quick install

1. **Unzip** the file you received (you'll get a `chrome-extension` folder).
2. Open Chrome → **`chrome://extensions`**.
3. Turn on **Developer mode** (top-right).
4. Click **Load unpacked** → select the **`chrome-extension`** folder (the one with `manifest.json`).
5. Pin the **Rep Pulse** icon (puzzle-piece 🧩 → pin).

Then: open any Reputation app page, use it, click **Rep Pulse → Download report**, and share the `.html`.

**Full guide:** see [README.md](README.md) — what it does, what it measures, how to read the report, and troubleshooting.

> Updating to a new version? On `chrome://extensions`, click **Reload** on the Rep Pulse card.
